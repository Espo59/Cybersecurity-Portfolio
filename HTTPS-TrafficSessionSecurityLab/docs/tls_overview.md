# 🔐 TLS / HTTPS Overview

## What is TLS?

Transport Layer Security (TLS) is a cryptographic protocol designed to provide:

- Confidentiality
- Integrity
- Authentication

for network communications.

---

## HTTPS

HTTPS combines:

- HTTP (application protocol)
- TLS (encryption layer)

to secure web traffic.

---

## Handshake Process

1. Client connects to server
2. Server presents certificate
3. Encryption keys are negotiated
4. Secure communication begins

---

## Importance

TLS protects against:

- Packet sniffing
- Man-in-the-Middle attacks
- Session interception
